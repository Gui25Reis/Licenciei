Fluxo Principal
Ações do Ator. Ações do Sistema.
1. Empresa seleciona o produto. 2. Sistema retorna as informações do produto.
3. Empresa solicita a edição do produto. 4. Sistema pede os dados que serão alternados.
5. Sistema válida os dados.
6. Sistema modifica dados do produto.
7. Sistema manda um email para o usuário notificando sobre a alteração feita.
